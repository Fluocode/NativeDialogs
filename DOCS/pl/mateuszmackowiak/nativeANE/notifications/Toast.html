public static function showWithDifferentGravit(message:String, duration:int = 0x00000001, gravity:int = 0, xOffset:int = 0, yOffset:int = 0):void

Make a standard toast that just contains a text with the text from a resource.

Parameters

message:String — the text displayed on the Toast
 
duration:int (default = 0x00000001) — How long to display the message. Either LENGTH_SHORT or LENGTH_LONG
 
gravity:int (default = 0) — sets the location at which the notification should appear on the screen.(GRAVITY_BOTTOM , GRAVITY_CENTER,...) (only on Android)
 
xOffset:int (default = 0) — the x offset from the gravity point (only on Android)
 
yOffset:int (default = 0) — the y offset from the gravity point (only on Android)


Throws
Error — if the call was unsuccessful.
