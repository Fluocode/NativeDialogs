dispose()method
public function dispose():void

Disposes of this ExtensionContext instance. The runtime notifies the native implementation, which can release any associated native resources. After calling dispose(), the code cannot call the call() method and cannot get or set the actionScriptData property.


Throws
Error — if the call was unsuccessful. Or will dispatch an Error Event.ERROR if there is a listener.
