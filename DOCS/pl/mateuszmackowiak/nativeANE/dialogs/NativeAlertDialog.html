show()method 
public static function show(message:String, title:String = Error, closeLabel:String = OK, otherLabels:String, closeHandler:Function = null, cancelable:Boolean = true, theme:int = -1):NativeAlertDialog

Creates and shows a NativeAlertDialog dialog. Use event.prevetDefault() to prevent disposing.

Parameters

message:String — message showed in the Alert control.
 
title:String (default = Error) — title of the Alert control.
 
closeLabel:String (default = OK) — text of the default button
 
otherLabels:String — text of the other buttons. Sepperated with "," adds aditional buttons. In the close event answer look for the index of the button.
 
closeHandler:Function (default = null) — function called when dialog closes (NativeDialogEvent). Called only once and if not event.preventDefault() will be removed.
 
cancelable:Boolean (default = true) — on back button or outside relese closes with index -1 (only Android)
 
theme:int (default = -1) — default -1 uses the defaultAndroidTheme

Returns
NativeAlertDialog

Throws
Error — if the call was unsuccessful.
